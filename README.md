# Blog

Building a blog with Evidence. 


## Getting Started


```bash

npm install
npm run sources
npm run dev

```


## Template Layout

The `+layout.svelte` file in `pages` allows to change the layout of the template.

It was used to change the `logo` and `footer`. But also could be used to add analytics library.
